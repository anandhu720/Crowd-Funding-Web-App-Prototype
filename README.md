## Installation

This web app requires [Node.js](https://nodejs.org/) v10+ to run.

Install the dependencies and devDependencies.
- Install Metamask in your browser
- Make account in Metamask
- Use Rinkeby test network


```sh
#In terminal
  npm i
  npm run dev
```
